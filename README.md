# Umbraflow

A sleek, dark theme for **Discord** and **Garry's Mod**, built on the Windows 11 **Fluent** design language. Smooth surfaces, subtle depth, and a cohesive look across both platforms.

> Made from scratch. Inspired by Windows 11 Fluent Design Language.

---

## ✨ Features

- **Fluent-inspired design** — acrylic/mica-style surfaces, rounded corners, and soft shadows drawn from Windows 11.
- **Consistent across platforms** — the same visual identity whether you're chatting or in-game.
- **Dark-first** — tuned for low-light comfort without crushing contrast.
- **Lightweight** — no bloat, just the styling.

## 🎮 Installation

### Discord

Umbraflow works with any injector that supports custom themes. Pick the one you use:

<details>
<summary><strong>BetterDiscord</strong></summary>

1. Install [BetterDiscord](https://betterdiscord.app/) if you haven't already.
2. Download `Umbraflow.theme.css` from this repo (or the [Releases](../../releases) page).
3. Open Discord → **Settings** → **Themes** → **Open Themes Folder**.
4. Drop `Umbraflow.theme.css` into that folder.
5. Toggle **Umbraflow** on in the Themes list.

</details>

<details>
<summary><strong>Vencord</strong></summary>

1. Install [Vencord](https://vencord.dev/).
2. Open Discord → **Settings** → **Themes** → **Open Themes Folder**.
3. Place `Umbraflow.theme.css` (or `.css`) into that folder.
4. Enable it in the Themes list.

> Alternatively, use the **Online Themes** field with a raw link to the CSS file.

</details>
